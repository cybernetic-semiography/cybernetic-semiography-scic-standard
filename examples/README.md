# SCIC Examples

This directory contains code examples and Jupyter notebooks demonstrating the Cybernetic Semiography (SCIC) framework.

## Quick Start Examples

### Basic Examples (Python Scripts)

1. **[01_basic_glyph_creation.py](01_basic_glyph_creation.py)** - Create and render a simple glyph
2. **[02_agent_activation.py](02_agent_activation.py)** - Demonstrate agent activation and inhibition
3. **[03_k_line_memory.py](03_k_line_memory.py)** - Memory snapshot and recall
4. **[04_agency_composition.py](04_agency_composition.py)** - Build complex agencies from agents
5. **[05_full_cycle_demo.py](05_full_cycle_demo.py)** - Complete R→I→A→E→X operational cycle

### Interactive Notebooks

Located in `notebooks/`:

1. **[Getting_Started.ipynb](notebooks/Getting_Started.ipynb)** - Interactive tutorial for newcomers
2. **[Minskyan_Architecture.ipynb](notebooks/Minskyan_Architecture.ipynb)** - Deep dive into Society of Glyphs
3. **[Patent_Proof_Walkthrough.ipynb](notebooks/Patent_Proof_Walkthrough.ipynb)** - Demonstrates patent claims

## Running Examples

```bash
# Run a basic example
python examples/01_basic_glyph_creation.py

# Run Jupyter notebooks
jupyter notebook examples/notebooks/
```

## Prerequisites

```bash
pip install -r ../requirements.txt
```
